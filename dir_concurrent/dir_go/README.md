# Go Concurrent Programming Projects

This directory contains Go projects related to concurrent programming.

## Subdirectories

*   [bakery-problem](https://github.com/unseen2004/studies/tree/main/dir_concurrent/dir_go/bakery-problem) - An implementation of Lamport's bakery algorithm.
*   [szymanski-algorithm](https://github.com/unseen2004/studies/tree/main/dir_concurrent/dir_go/szymanski-algorithm) - An implementation of Szymanski's algorithm for mutual exclusion.
