# Pascal's Triangle Visualizer (Swing)

This directory contains a simple Java Swing application for visualizing Pascal's Triangle.

## Description

The application provides a graphical user interface where you can enter an integer `n`. After clicking the "run" button, the application will display the first `n` rows of Pascal's Triangle. The window resizes dynamically to accommodate the size of the triangle.

## How to Run

To compile and run this application, you can use the `javac` and `java` commands from your terminal.

1.  **Compile the code:**
    Navigate to the `PascalTriangleVisualizer` directory and run:
    ```bash
    javac Main.java
    ```

2.  **Run the application:**
    ```bash
    java Main
    ```
