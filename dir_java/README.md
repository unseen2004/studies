# Java Projects

This directory contains a collection of projects written in Java.

## Projects

*   [gui](https://github.com/unseen2004/studies/tree/main/dir_java/gui) - A simple GUI application.
*   [javafxPascalTriangle](https://github.com/unseen2004/studies/tree/main/dir_java/javafxPascalTriangle) - A visualization of Pascal's Triangle using JavaFX.
*   [javaSimplePaint](https://github.com/unseen2004/studies/tree/main/dir_java/javaSimplePaint) - A simple paint program.
*   [PascalTriangleVisualizer](https://github.com/unseen2004/studies/tree/main/dir_java/PascalTriangleVisualizer) - A command-line visualizer for Pascal's Triangle.
*   [server](https://github.com/unseen2004/studies/tree/main/dir_java/server) - A simple client-server application.
*   [shapeCalculator](https://github.com/unseen2004/studies/tree/main/dir_java/shapeCalculator) - A calculator for the area and perimeter of different shapes.
