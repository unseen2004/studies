# JavaFX Pascal's Triangle Visualizer

This directory contains a JavaFX application that visualizes Pascal's Triangle.

## Description

The application provides a graphical interface where the user can enter a number `n`, and the application will display the first `n` rows of Pascal's Triangle. The window size adjusts dynamically to fit the generated triangle.

## How to Run

This is a Maven project, so you can build and run it using the following commands:

1.  **Build the project:**
    ```bash
    mvn clean install
    ```
2.  **Run the application:**
    ```bash
    mvn javafx:run
    ```
