# Rust Projects

This directory contains a collection of projects written in Rust.

## Projects

*   [minigrep](https://github.com/unseen2004/studies/tree/main/dir_rust/minigrep) - A simple, command-line text searching tool, similar to `grep`.
*   [pattern_search](https://github.com/unseen2004/studies/tree/main/dir_rust/pattern_search) - A collection of string searching algorithms.
*   [queue_system](https://github.com/unseen2004/studies/tree/main/dir_rust/queue_system) - A simulation of a queueing system.
*   [rust_cli](https://github.com/unseen2004/studies/tree/main/dir_rust/rust_cli) - A template or example of a Rust command-line interface (CLI) application.
